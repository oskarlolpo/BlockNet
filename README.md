## 📝 Описание

**BlockNet** — это современное десктопное приложение, которое позволяет игрокам легко объединяться в локальных мирах Minecraft через интернет. Забудьте о сложностях с пробросом портов (Port Forwarding), настройке Hamachi или Radmin VPN. BlockNet использует современные P2P-технологии и протокол **QUIC** для прямого, быстрого и безопасного соединения между компьютерами.

Приложение выступает как мост: хост открывает свой мир для локальной сети, BlockNet публикует его в глобальном матчмейкинг-лобби, а другие игроки подключаются к этому миру буквально в два клика.

## ✨ Ключевые возможности

- 🌍 **Глобальное лобби:** Просматривайте список доступных миров и серверов в реальном времени.
- ⚡ **Прямое P2P-соединение (QUIC):** Обход NAT с помощью STUN для обеспечения минимального пинга без сторонних серверов.
- 📊 **Живой статус игроков:** Приложение автоматически пингует локальный сервер и показывает реальное количество игроков (например, `1/30`) прямо в глобальном лобби.
- 🛡 **Безопасность:** Весь трафик шифруется. Возможность устанавливать пароли на комнаты.
- 🎮 **Поддержка Java и Bedrock Edition.**
- 🚀 **Автоматическая сборка:** Интеграция с GitHub Actions для автоматической компиляции `.exe` и `.msi` установщиков.

## 🛠 Технологический стек

- **Backend:** [Rust](https://www.rust-lang.org/) + [Tauri](https://tauri.app/)
  - Сеть: `tokio`, `quinn` (QUIC протокол), `async-minecraft-ping`
- **Frontend:** [Vue.js 3](https://vuejs.org/) (Composition API) + [Vite](https://vitejs.dev/)
- **Дизайн:** Custom CSS, современный темный UI в стиле Glassmorphism

## ⚙️ Установка и Сборка

Вы можете скачать готовый установщик (`.exe` или `.msi`) в разделе **[Releases](../../releases)** на GitHub. 

Если вы хотите собрать проект самостоятельно:

### Требования
- Node.js (v20+)
- Rust (stable)
- Установленные Build Tools для Windows (Visual Studio C++ Build Tools)

### Инструкция
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/oskarlolpo/BlockNet.git
   cd BlockNet
   ```
2. Установите зависимости frontend-части:
   ```bash
   npm install
   ```
3. Запустите в режиме разработчика:
   ```bash
   npm run tauri dev
   ```
4. Соберите релизную версию:
   ```bash
   npm run tauri build
   ```

## 🎮 Как использовать

**Для Хоста (Создателя сервера):**
1. Зайдите в Minecraft и откройте мир для сети (или запустите выделенный сервер).
2. Запустите BlockNet, нажмите **"Создать хост"**.
3. Укажите локальный порт, который выдал Minecraft (например, `56571`).
4. Приложение пробьет NAT и опубликует ваш сервер в глобальном лобби.

**Для Клиента (Игрока):**
1. Откройте BlockNet.
2. Найдите нужный сервер в списке лобби и нажмите **"Подключиться"**.
3. Приложение создаст локальный туннель (например, `127.0.0.1:45300`).
4. Зайдите в Minecraft -> Сетевая игра -> Добавить сервер -> введите адрес, который выдал BlockNet.
5. Наслаждайтесь игрой!
